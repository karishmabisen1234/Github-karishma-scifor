HTML Notes

What is HTML?

HTML stands for HyperText Markup Language which is used to create a webpage.

What are the main features of HTML?

It is used for creating the web browser by using different tags. We can add images and videos in the web browser with the help of HTML.

What are HTML Tags?

HTML Tags are the format by which we can give the instructions how we have to display our content.

There are two kinds of tags in HTML:

  1. Paired tags: These are tags which are in the form of pair.
    Example: <p></p>, <div></div>
  2. Self closing or Non-paired tags: These are tags which are not in pair.
    Example: <br>, <img>, <hr>
