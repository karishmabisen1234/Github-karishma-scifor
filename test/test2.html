CSS Notes

1. What is CSS and Why Do We Use It in Web Development?

CSS stands for Cascading Style Sheets. It is used to design and style a webpage created with HTML. CSS defines how elements will appear — including layout, color, font, and positioning — to make webpages visually attractive and organized.

2. Difference Between Inline, Internal, and External CSS (with Examples)

I) Inline CSS: CSS that is written directly within the HTML element using the style attribute.

Example:

<html>
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    <h1 style="color: blue">This is a HTML page</h1>
</body>
</html>

II) Internal CSS: CSS that is written inside the <style> tag within the <head> section of an HTML document.

Example:

<html>
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        div {
            background-color: lightgray;
            color: blue;
        }
    </style>
</head>
<body>
    <div>
        Hello this is test
    </div>
</body>
</html>

III) External CSS: CSS that is written in a separate .css file and linked to an HTML document using the <link> tag. This method is best for large websites or multiple pages.

Example:

<html>
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <h1>This is a web page</h1>
</body>
</html>

3. Define the CSS Box Model and Describe Its Components

The CSS Box Model represents how elements are structured and how their size is calculated. It includes the total height and width of an element.

Components of Box Model:

  • Margin: The outer space of the element (outside the border).
  • Border: The line between margin and padding.
  • Padding: The space between the content and border (inner space).
  • Content: The actual text, image, or element inside the box.

4. Difference Between ID and Class Selectors in CSS

  • ID: Used for styling a single unique element. It cannot be reused. Selected using # symbol.
    Example: #title { color: red; }
  • Class: Used for styling multiple elements (group). It can be reused. Selected using . symbol.
    Example: .box { background-color: yellow; }
